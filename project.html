Project Details

Project Overview

This project develops an automated glaucoma detection system using deep learning to analyze retinal fundus images for early signs of glaucoma, a leading cause of irreversible blindness worldwide.

The system combines advanced computer vision techniques with a custom convolutional neural network to provide accurate and fast screening capabilities that can assist ophthalmologists in early diagnosis.

Technologies Used

  • Python with TensorFlow/Keras
  • Flask Web Framework
  • Bootstrap 5
  • OpenCV Image Processing

Key Features

Advanced CNN Architecture

Custom 4-layer convolutional neural network with batch normalization and dropout layers for effective regularization and stable training.

Comprehensive Dataset

Trained on 10,000+ retinal images from multiple public datasets including ACRIMA, ORIGA, and RIM-ONE with extensive data augmentation.
